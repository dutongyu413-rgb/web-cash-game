# 事件卡池与身份规则

> 更新日期：2026-07-22
> 对应版本：`0.4.12-internal`

当前正式卡池分为三层：全身份公共事件、职业组共享事件、单职业专属事件。职业只影响工作语义相关卡牌；家庭、健康、住房和消费仍属于随机人生事件。市场先生报价是卡池之外的独立月度系统。

## 一、全身份公共事件（配置47张，当前启用46张）

这些事件不依赖具体职业，所有预设身份和自定义身份都可以遇到。

| 类型 | 事件 |
| --- | --- |
| 家庭与长期支出 | 孩子报课外班、房租上涨、父母生活支持增加、育儿支出增加、亲戚来住一阵、临时请人照护、夏季电费上涨、通勤成本增加 |
| 一次性生活支出 | 老人住院、车辆维修、搬家支出、某个家电出现故障、手机突然报废、集中人情往来、家里漏水维修、朋友婚礼出行、宠物看病、电脑维修、节日开销 |
| 收入与现金机会 | 短期失业、副业收入、控制消费成功、退税或补贴到账、朋友介绍兼职单、闲置物品卖出、取消闲置订阅、旧押金退回、积分抵扣账单 |
| 健康与保障 | 自己生病休养、照顾家人、牙齿治疗、医保报销后的自付账单、孩子反复发烧、运动受伤、检查保障配置 |
| 生活选择与兴趣 | 健身年卡续费、离公司近一点、带家人短途旅行、带孩子去游乐场、周末看电影聚餐、报名兴趣课程、入门钓鱼装备、周末户外徒步、瑜伽体验课、是否换车 |
| 财务与成长 | 职业提升课程；建立应急储备计划暂时隐藏，仅保留配置 |

说明：游戏没有婚育、宠物或住房状态字段，因此这些生活事件暂不做身份硬过滤。后续只有增加了对应身份数据，才适合进一步拆池。

收入中断采用最高优先级：短期失业生效期间常规月收入为零；职业课程回本会顺延到收入恢复后，一次性现金收入不受影响。

## 二、职业限定或共享事件（12张）

这类事件必须满足工作方式前提。自定义身份会保留一般工作事件，但运动员商业机会和咖啡店经营事件属于严格职业限定，不进入自定义身份卡池。

| 事件 | 可以出现的身份 |
| --- | --- |
| 公司业务调整 | 企业白领、钢琴老师、高级工程师、数据分析师、建筑师、医生、程序员、图书管理员 |
| 绩效奖金缩水 | 企业白领、钢琴老师、销售、高级工程师、数据分析师、建筑师、医生、程序员、图书管理员 |
| 项目奖金延期 | 企业白领、自媒体博主、销售、高级工程师、数据分析师、建筑师、程序员、收纳师 |
| 提成到账变慢 | 销售 |
| 临时无薪假 | 企业白领、钢琴老师、销售、高级工程师、数据分析师、建筑师、医生、运动员、程序员、图书管理员 |
| 客户预算收缩 | 自媒体博主、销售、建筑师、收纳师 |
| 年终奖到账 | 企业白领、钢琴老师、销售、高级工程师、数据分析师、建筑师、医生、程序员、图书管理员 |
| 公司发放购物卡 | 企业白领、钢琴老师、销售、高级工程师、数据分析师、建筑师、医生、程序员、图书管理员 |
| 商务活动出席 | 运动员 |
| 品牌代言邀约 | 运动员 |
| 邀请探店博主 | 咖啡主理人 |
| 门口来了一只流浪猫 | 咖啡主理人 |

运动员不会再遇到客户预算收缩、提成到账变慢、公司业务调整、绩效奖金缩水、年终奖到账或公司购物卡。

运动员的商务活动权重高于品牌代言；普通副业和朋友介绍兼职单仍可出现，但概率降为默认职业的75%。

咖啡主理人的两张经营事件不会进入其他身份卡池；流浪猫选择救助后，三个月后会随机形成猫咖收入提升或幼猫照料支出。

## 三、单职业专属事件（14张）

专属事件只对绑定身份出现，并在一局第2至第5个月触发一次。其他身份不会进入这些卡牌的候选池。

| 身份 | 专属事件 |
| --- | --- |
| 企业白领 | 跨部门项目邀请 |
| 自媒体博主 | 平台商务合作 |
| 餐饮老板 | 后厨设备升级 |
| 咖啡主理人 | 商圈客流变少 |
| 钢琴老师 | 成人钢琴小班 |
| 销售 | 大客户订单 |
| 高级工程师 | 核心系统升级 |
| 数据分析师 | 临时数据项目 |
| 建筑师 | 方案竞标 |
| 医生 | 进修名额 |
| 运动员 | 赛季装备更新 |
| 程序员 | 线上故障值班 |
| 收纳师 | 全屋收纳加单 |
| 图书管理员 | 周末阅读活动 |

14个预设身份均已有一张单职业专属事件；自定义身份没有职业标签，因此只使用公共事件和职业组共享事件。

## 四、概率规则

- 硬资格先排除语义明显不成立的卡牌。
- 其余候选卡继续随机，不做固定剧情和普通卡保底。
- 高相关事件使用 `1.35` 倍权重，较相关事件使用 `1.20` 倍，弱相关事件使用 `0.75` 倍。
- 同一局仍然不会重复抽到同一张主事件卡。
- 游戏与数值模拟器共同读取 `career-event-rules.js`，避免测试规则和正式规则不一致。

## 五、卡池之外的市场报价

“市场先生报价”不是生活事件卡，不占用当月主事件，也不受“同局主事件不重复”限制。市场净值每月独立运行，自动报价按以下顺序判断：

1. 净值实际下跌时不自动弹出。
2. 净值连续上涨3个月时强制弹出一次，触发后同一上涨链不重复。
3. 其他上涨或震荡月份按20%概率随机弹出。

玩家仍可在任意月份从地图手动查看当月报价；手动查看不会重新生成净值。
